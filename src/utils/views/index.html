Bienvenido al backend de huertos

Con este back-end podras registrar y obtener informaci贸n sobre tus huertos. Estos son sus diferentes end-points.

End points para los usuarios /api/v1/user
POST /api/v1/user/register - Con este end point podras registrarte como usuario.
BODY

{

"nick": "pepito",

"password": "pepito123"

}

POST /api/v1/user/login - Con este end point podras ingresar como usuario de la plataforma.
BODY

{

"nick": "pepito",

"password": "pepito123"

}

POST /api/v1/user/register-huerto - Con este end point podras registrar los huertos de un usuario.
HEAD

{

"Autorization": "bearer token-authentication"

}

BODY

{

"name": "bearer token-authentication"

}

GET /api/v1/user/huertos/:limit/:page - Con este end point podras listar los diferentes huertos de un usuario.
HEAD

{

"Autorization": "bearer token-authentication"

}

End points para los arduinos /api/v1/arduino
POST /api/v1/arduino/register-climate
HEAD

BODY
